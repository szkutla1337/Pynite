What's Pynite?

Pynite is an library made in Pygame which is used to create window applications and provide user with simple ways to draw shapes and create useful things such as buttons, slider bars, check boxes, animations, text boxes etc. which you can freerly customize. It can also be used to create 2D games. Pynite has also some useful classes and functions about which you can learn more in the documentaion.

What libraries does Pynite use?

  • Pygame
  • Pyperclip
  • Numpy
  • Pywin32
  • Mutagen
