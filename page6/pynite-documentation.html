Learn about attributes of widgets in Pynite

x - places button on the screen in 'x' direction <int>

y - places button on the screen in 'y' direction <int>

background - changes color of the button, e.g: (r, g, b) <tuple>

hoverBackground - changes color of the button when mouse is hovering over it, e.g: (r, g, b) <tuple>

activeBackground - changes color of the button when the button is clicked, e.g: (r, g, b) <tuple>

text changes text that's displayed in the button <str>

textColor changes color of the text that's displayed in the button, e.g: (r, g, b) <tuple>

hoverTextColor changes color of the text when mouse is hovering over it, e.g: (r, g, b) <tuple>

activeTextColor changes color of the text when the button is clicked, e.g: (r, g, b) <tuple>

outlineWidth - changes width of an outline around the button <int>

outlineColor - changes color of the outline, e.g: (r, g, b) <tuple>

hoverOutlineColor - changes color of the outline when mouse is hovering over it, e.g: (r, g, b) <tuple>

activeOutlineColor - changes color of the outline when the button is clicked, e.g: (r, g, b) <tuple>

fontName - changes font of the text displayed in the button, e.g: ".ttf" or "Arial" <str>

fontSize - changes size of the text displayed in the button <int>

autoResizing - determines if the button resizes automatically based on the length of text <bool>

width - changes width of the button (when autoResizing is set to true then it adds specified value to current width) <int>

height - changes height of the button (when autoResizing is set to true then it adds specified value to current height) <int>

changeCursor - determines if cursor should change to "pointer" whenever the mouse is hovering over the button <bool>

borderRadius - sets radius of the button border <int>

bordersToHide - hides roundness of specified corners, e.g: ["topleft", "bottomright"] <list>

clickAnimation - determines if the button should "jump" or stay static when clicked <bool>

hide - determines if button should be rendered <bool>

disable - disables/enables the button (when false the button won't work) <bool>

textPosition - places text of the button in specified position, e.g: "center", "left", "right" <str>

textOffsetX - offsets text position in 'x' direction <int>

textOffsetY - offsets text position in 'y' direction <int>

x - places label on the screen in 'x' direction <int>

y - places label on the screen in 'y' direction <int>

text - changes text that's being displayed <str>

textColor - changes color of the text that's displayed, e.g: (r, g, b) <tuple>

background - changes background color of the label (default: transparent), e.g: (r, g, b) <tuple>

backgroundWidth - changes width of the background <int>

backgroundHeight - changes height of the background <int>

charSpacing - adjusts spacing between the letters (default: 0) <int>

charColor - changes color of individual letters that have been specified, e.g: [(index, (r,g,b))] <list>

hide - determines if label should be rendered <bool>

fontName - changes font of the text, e.g: ".ttf" or "Arial" <str>

fontSize - changes size of the text <int>

x - places rectangle on the screen in 'x' direction <int>

y - places rectangle on the screen in 'y' direction <int>

background - changes color of the rectangle, e.g: (r, g, b) <tuple>

width - changes width of the rectangle <int>

height - changes height of the rectangle <int>

thickness - changes thicknes of the rectangle (default: 0) <int>

borderRadius - sets radius of the rectangle border <int>

hide - determines if rectangle should be rendered <bool>

x - places textbox on the screen in 'x' direction <int>

y - places textbox on the screen in 'y' direction <int>

background - changes background color of the textbox, e.g: (r, g, b) <tuple>

activeBackground - changes color of the button when the textbox is active, e.g: (r, g, b) <tuple>

width - changes width of the textbox <int>

height - changes height of the textbox <int>

borderRadius - sets radius of the textbox border <int>

maxCharacters - sets maximum amount of characters that can be typed into textbox <int>

numbersOnly - when true, only numbers can be written to the textbox <bool>

lettersOnly - when true, only letters can be written to the textbox <bool>

exceptionChars - determines which characters can be written tot the textbox, e.g: ["v", "g", "s", "r"] <list>

text - changes text that's being displayed in the textbox <str>

placeholderText - sets placeholder text in the textbox to specified value <str>

textColor - changes color of the text that's displayed in the textbox, e.g: (r, g, b) <tuple>

placeholderColor - changes color of the placeholder text that's displayed in the textbox, e.g: (r, g, b) <tuple>

changeCursor - determines if cursor should change to "IBeam" whenever the mouse is hovering over the textbox <bool>

fontName - changes font of the text, e.g: ".ttf" or "Arial" <str>

fontSize - changes size of the text <int>

cursorColor - changes color of the blinking cursor that indicates current position in the textbox, e.g: (r, g, b) <tuple>

showOutline - determines if outline of the textbox should be rendered <bool>

disable - disables/enables the textbox (when false the textbox won't work) <bool>

outlineWidth - changes width of an outline around the textbox <int>

outlineColor - changes color of the outline, e.g: (r, g, b) <tuple>

hoverOutlineColor - changes color of the outline when mouse is hovering over it, e.g: (r, g, b) <tuple>

activeOutlineColor - changes color of the outline when the textbox is active, e.g: (r, g, b) <tuple>

highlightColor - changes the text selection color, e.g: (r, g, b) <tuple>

bold - determines if text sould be bold <bool>

hide - determines if textbox should be rendered <bool>

x - places image on the screen in 'x' direction <int>

y - places image on the screen in 'y' direction <int>

image - changes image that's being displayed (takes path to the image) <str>

scale - changes scale of the image (default: 1) <int>

fill - fills the transparent parts of the image with specified color, e.g: (r, g, b) <tuple>

hide - determines if image should be rendered <bool>

x - places imagebutton on the screen in 'x' direction <int>

y - places imagebutton on the screen in 'y' direction <int>

image - changes image that's being displayed (takes path to the image) <str>

image2 - changes image that's being displayed when mouse is hovering over the imagebutton (takes path to the image) <str>

scale - changes scale of the image (default: 1) <int>

fill - fills the transparent parts of the image with specified color, e.g: (r, g, b) <tuple>

changeCursor - determines if cursor should change to "pointer" whenever the mouse is hovering over the imagebutton <bool>

clickAnimation - determines if the imagebutton should "jump" or stay static when clicked <bool>

disable - disables/enables the imagebutton (when false the button won't work) <bool>

hide - determines if imagebutton should be rendered <bool>
